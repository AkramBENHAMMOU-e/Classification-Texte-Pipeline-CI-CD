# Projet MLOPS Classification Text CI/CD

Ce projet a pour but de mettre en place une pipeline MLOps complète pour la classification de texte, incluant l'intégration continue et le déploiement continu (CI/CD).

## Structure du projet

- `data/`: Données du projet
- `src/`: Code source
- `tests/`: Tests unitaires et d'intégration
- `models/`: Modèles entraînés
- `.github/workflows/`: Workflows GitHub Actions
